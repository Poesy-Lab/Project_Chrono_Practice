04 Chrono 기본 구성

PyChrono 핵심 5단계 패턴

# ① 시스템 생성
sys = chrono.ChSystemNSC()
sys.SetGravitationalAcceleration(chrono.ChVector3d(0, -9.81, 0))

# ② 물체 추가
body = chrono.ChBodyEasySphere(0.5, 1000, True, True, material)
sys.AddBody(body)

# ③ 조인트 연결
joint = chrono.ChLinkRevolute()
joint.Initialize(body_A, body_B, frame)
sys.Add(joint)

# ④ 시각화 (VSG 우선, Irrlicht 폴백 — 자동 분기)
try:
    import pychrono.vsg3d as chronovsg
    vis = chronovsg.ChVisualSystemVSG()
except ImportError:
    vis = chronoirr.ChVisualSystemIrrlicht()

# ⑤ 시뮬레이션 루프
while vis.Run():
    vis.BeginScene(); vis.Render(); vis.EndScene()
    sys.DoStepDynamics(0.005)

시각화 시스템 비교: Irrlicht vs VSG

기능Irrlicht (OpenGL)VSG (Vulkan) ★
텍스처 / 재질macOS 제한적정상 렌더링
Retina 디스플레이1/4만 렌더링전체 화면
선(line) 렌더링macOS 안 됨정상
GUI 패널없음ImGui 통합
추가 빌드 필요?아니오예 (선택사항)
코드에서 try: import pychrono.vsg3d 패턴으로 VSG 자동 선택. Windows/Linux 팀원은 Irrlicht로 정상 동작, macOS는 VSG 권장.

스프링-댐퍼: Force Functor 패턴

class MySpringForce(chrono.ForceFunctor):
    def evaluate(self, time, rest_length,
                 length, vel, link):
        Fs = -k * (length - rest_length)
        Fd = -c * vel
        Fext = F_amp * np.sin(F_freq * time)
        return Fs + Fd + Fext
