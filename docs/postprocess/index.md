# Postprocess — 후처리/데이터 출력

> 🟢 우리 팀 필수 (Phase 4) | 💻 CPU만으로 동작
> 📖 [공식 매뉴얼](https://api.projectchrono.org/group__postprocess__module.html)

시뮬레이션 결과를 파일(이미지, 영상, 그래프)로 출력합니다.
CSV/JSON 출력은 Python 표준 라이브러리로 직접 구현합니다.

## 주요 기능

| 기능 | 클래스 | 설명 |
|---|---|---|
| POV-Ray | `ChPovRay` | 레이트레이싱 애니메이션 스크립트 생성 |
| GNUplot | `ChGnuPlot` | 그래프 자동 생성 |
