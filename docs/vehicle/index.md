# Vehicle — 차량/로버 시뮬레이션

> 🟢 우리 팀 필수 | 💻 CPU만으로 동작
> 📖 [공식 매뉴얼](https://api.projectchrono.org/manual_vehicle.html)

템플릿 기반 설계로, 서스펜션/타이어/구동계 등을 조합하여 차량을 구성합니다.
C++ 코딩 없이 **JSON 파일만으로** 차량 파라미터를 정의할 수도 있습니다.
