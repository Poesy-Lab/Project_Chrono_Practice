# Sensor — 센서 시뮬레이션

> 🟡 확장 모듈 | 🎮 NVIDIA GPU + CUDA + OptiX 필요
> 📖 [공식 매뉴얼](https://api.projectchrono.org/manual_sensor.html)

가상 센서(카메라, LiDAR, GPS, IMU 등)를 시뮬레이션합니다.
자율주행/로봇 감지 파이프라인에 활용됩니다.

## 센서 종류

| 센서 | 설명 |
|---|---|
| Camera | RGB/깊이 이미지 (레이트레이싱) |
| LiDAR | 3D 포인트 클라우드 |
| GPS | 위치 좌표 |
| IMU | 가속도계 + 자이로스코프 |
| Radar | 레이더 반사 |
| Tachometer | 회전 속도 |
